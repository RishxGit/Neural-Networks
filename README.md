# Neural Networks 

This repository serves as a personal practice space for learning the foundational mechanics of Neural Networks and Deep Learning.

## 🧠 What's Inside
* **`Neural_Networks.ipynb`**: Core notebook detailing network architecture, forward propagation, and training.
* **Multiclass Classification**: Workflows dedicated to multi-category sorting, featuring a **Handwritten Digits Classification** project.

## 🛠️ Concepts Practiced
Forward propagation, Softmax activation, Multiclass cost functions, and Neural Network training loops.
